Trust Dashboard — Hushh Hackathon

What is this?
A prototype dashboard that tracks how user data 
is being used across apps and devices.

Problem it solves
Users have no visibility into which apps are 
accessing their data and where it is going.

Features
- Tracks which apps access user data
- Shows where data is going and who can see it
- Notifies users about suspicious data sharing
- Allows users to block harmful data access
- Hardware-level device tracking

 Tech Stack
- Frontend: React/HTML/CSS
- Backend: Python
